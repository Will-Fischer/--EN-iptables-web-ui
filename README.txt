This code is an Web interface for iptables, the Linux firewall.

It allows you to do a multitude of things such as:

    Block an IP address
    Allow an IP address
    Allow ping
    Block ping
    Block a port
    Allow a port
    Allow UDP
    Block UDP
    Clear rules
    Block external traffic


You should have an SSH server, and activate SSH2 add-on on your PHP server
